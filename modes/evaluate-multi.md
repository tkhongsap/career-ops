# Mode: evaluate-multi — เปรียบเทียบข้อเสนองานหลายรายการ

**ภาษา:** ผู้ใช้เขียนภาษาไทย → ตอบภาษาไทย | ผู้ใช้เขียนภาษาอังกฤษ → ตอบภาษาอังกฤษ

Scoring matrix 10 มิติแบบถ่วงน้ำหนัก:

| มิติ | น้ำหนัก | เกณฑ์ 1–5 |
|------|---------|-----------|
| ความสอดคล้องกับเป้าหมาย | 25% | 5=ตรงกับ target role ทุกประการ, 1=ไม่เกี่ยวข้อง |
| จับคู่กับ CV | 15% | 5=match 90%+, 1=match <40% |
| ระดับ (senior+) | 15% | 5=staff+, 4=senior, 3=mid-senior, 2=mid, 1=junior |
| เงินเดือนประมาณการ (฿/เดือน) | 10% | 5=top quartile ของตลาด, 1=ต่ำกว่าตลาด |
| เส้นทางการเติบโต | 10% | 5=มีเส้นทางชัดเจน, 1=ทางตัน |
| คุณภาพการทำงานระยะไกล | 5% | 5=full remote async, 1=onsite เท่านั้น |
| ชื่อเสียงบริษัท | 5% | 5=employer ชั้นนำ, 1=มี red flags |
| ความทันสมัยของ tech stack | 5% | 5=cutting edge, 1=legacy |
| ความเร็วในการรับข้อเสนอ | 5% | 5=กระบวนการรวดเร็ว, 1=6+ เดือน |
| สัญญาณวัฒนธรรมองค์กร | 5% | 5=builder culture, 1=ราชการ/bureaucratic |

สำหรับแต่ละข้อเสนอ: คะแนนในแต่ละมิติ, คะแนนถ่วงน้ำหนักรวม
การจัดอันดับสุดท้าย + คำแนะนำพร้อมพิจารณา time-to-offer

ขอข้อเสนองานจากผู้ใช้ถ้ายังไม่มีใน context อาจเป็นข้อความ, URLs, หรืออ้างอิงข้อเสนอที่ประเมินแล้วใน tracker
